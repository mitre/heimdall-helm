## Configuration Methods

Environment variables can be set using three methods (in order of precedence):

::steps

### Existing Secret

Reference external secret management (External Secrets Operator, Vault, Sealed Secrets):

```yaml
heimdall:
  existingSecret: "my-heimdall-secrets"
```

### Inline Values

Set via values.yaml (for CI/CD):

```yaml
heimdall:
  config:
    GITHUB_CLIENTID: "abc123"
  secrets:
    GITHUB_CLIENTSECRET: "secret123"
```

### File-based

Use `env/heimdallconfig.yaml` and `env/heimdall-secrets.yaml` (for local development):

```bash
./generate-heimdall-secrets.sh
helm install heimdall ./heimdall --set-file heimdall.secretsFile=heimdall/env/heimdall-secrets.yaml
```
