## What's Included

### Security

- ClusterIP service (no external exposure by default)
- Ingress disabled by default (enable for HTTPS access)
- Security contexts for containers
- Three secrets management approaches (existing secret, file-based, inline)

### High Availability

- PodDisruptionBudget (default: minAvailable=1)
- Health probes: startup, liveness, readiness
- Rolling updates with configurable strategy
- HorizontalPodAutoscaler support

### Database

- Embedded PostgreSQL via Bitnami subchart (default)
- External database support (AWS RDS, GCP Cloud SQL, Azure Database)
- Migrations run automatically on application startup
- Admin user created from `ADMIN_EMAIL` and `ADMIN_PASSWORD` env vars

### Authentication

Multiple authentication methods:
- Local (email/password)
- LDAP
- GitHub OAuth
- GitLab OAuth
- Google OAuth
- Okta OIDC
- Generic OIDC

### Configuration

- Environment variables via ConfigMap
- Secrets via three approaches (see [Secrets Management](/getting-started/secrets))
- Values schema validation (JSON Schema)
- Helm 4 compatible (Server-Side Apply)
